# Project Management System

A comprehensive project management application with AI-powered features.

## Features

- **Workspaces & Projects** - Organize work in hierarchical structure
- **Kanban Board** - Drag-and-drop task management
- **AI Integration** - Auto-generate subtasks and summaries
- **Analytics** - Track team velocity and progress
- **Secure Auth** - Replit Auth integration

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: Express.js, Node.js
- **Database**: PostgreSQL with Drizzle ORM
- **AI**: OpenAI GPT-5.1
